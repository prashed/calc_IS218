SOLID

Object Oriented Design

SOLID

Single Responsibility

To do math operations.

Open Closed

When a math operation is done, add it to the history.

Liskov Substitution

Every Math operation class is built from the Operation class which holds the numbers to be calculated

Interface Segregation

We don’t want to import classes that aren’t needed. For example, we don’t want to import the Operation class, just the classes that are built from it: Addition, Subtraction, Multiplication, and Division.

Dependency Inversion

Write logic in a way that is scalable so that when circumstances or environments are different, the logic of the program will still work.

Create a python package whose name is "tests"

Then in terminal at the root of your project directory

Then create a python file within that folder with a prefix or suffix of “test_” or “_test

Within the python file, create functions with a prefix or suffix of “test_” or “_test”

When the program is run using the “pylint” command in terminal, pylint will look through the tests package and look for any functions with the “test” prefix or suffix and run those functions.
