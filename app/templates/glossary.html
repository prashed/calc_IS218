Glossary of Programming Terms

Instantiation

Creating an object.

Object

An instance of a class.

Class

A collection of related attributes and behaviors.

Namespace

Defining a section of a program where specific classes and functions are located.

Constructor

Sets up the core attributes of a given class.

Fixture

A fixture is a predetermined input value that can be run through similar testing processes.

Unit Test

Testing the flow of code to see if a given input results in an expected output.

Static

An attribute that should not be accessed directly, but through the logic of other functions.

Public

Attributes are available to all outside classes.

Private

Attributes are only available within the class.

Protected

Attributes are available to the class and its children

Method

A block of code that is responsible for a specific behavior and can be reused.

Property

An attribute of an object.
