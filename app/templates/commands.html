Commands You Need To Know:

Commands

Descriptions

pwd

present working directory

ls

list folders and files in the present directory

mkdir

make a directory (folder)

pressing "tab" twice

auto completes command or path

cd <directoryName>

change focus to specified directory

cd ..

go up one directory, go to direct parent directory

touch <fileName>

makes a file

rm <fileName>

remove/ delete a file

rm -r <directoryName>

remove a directory and all its contents

mv <fileName> <directoryName>

move a file

mv <fileName> <newFileName>

rename a file

Creating and Adding to a Git Repository:

git init

initialize repository

git add <fileName>

add file to staging area

git status

check what files are and are not in staging area

git commit -m "first commit"

commit all files in staging area with given message

git remote add origin <repositoryUrl>

add repository url with name "origin"

git push origin main

push all locally committed changes to origin
